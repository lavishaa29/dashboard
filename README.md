# Power BI Dashboards

Two Power BI dashboards covering business and content analytics — sales performance tracking and Netflix content trends.

## 1. Sales Performance Dashboard
`Sales_Performance_Dashboard.pbix`

An end-to-end sales analytics dashboard tracking individual and team performance against targets.

- Cleaned and modeled raw Excel sales data using Power Query
- KPI cards, trend charts, and salesperson-level sparklines
- DAX measures for variance and target-attainment tracking

## 2. Netflix Content Analytics Dashboard
`Netflix_Content_Analytics_Dashboard.pbix`

An interactive dashboard analyzing ~8,800 Netflix titles by type, genre, country, and release timeline.

- Built on a Kaggle dataset of Netflix titles
- Handled multi-value genre fields and missing data through Power Query transformations
- DAX measures for date-range filtering and content-growth analysis

## Tech stack

- **Power BI**
- **Power Query** (data cleaning + modeling)
- **DAX** (KPI, variance, and growth measures)

## How to view

Both files are `.pbix` — open them in [Power BI Desktop](https://www.microsoft.com/en-us/power-platform/products/power-bi/downloads) (free) to explore the dashboards interactively.
